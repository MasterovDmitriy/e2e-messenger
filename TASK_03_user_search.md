Реализуй фичу: User search by username

## Что нужно сделать
Добавь поиск пользователей по частичному совпадению username для авторизованных пользователей. Поиск должен исключать текущего пользователя и возвращать минимально необходимые публичные поля.

## Файлы которые нужно создать / изменить
- `backend/app/api/v1/users.py` — endpoint поиска
- `backend/app/schemas/user.py` — схема выдачи пользователя
- `backend/app/services/user_service.py` — бизнес-логика поиска
- `frontend/src/api/users.ts` — API-клиент поиска
- `frontend/src/features/user-search/*` — UI поиска и списка результатов

## Входные данные / интерфейс
`GET /api/v1/users/search?query=<string>&limit=<int>`, JWT обязателен. Возвращается массив пользователей с `id`, `username`, `public_key_fingerprint`.

## Ожидаемый результат
Пользователь может найти собеседника по username и выбрать его для открытия личного чата.
